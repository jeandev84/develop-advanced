# PSR-1, PSR-2, PSR-4

PSR-1: Basic Coding Standard
Данный стандарт описывает элементы,
являющиеся необходимыми для обеспечения технической совместимости кода,
создаваемого или поддерживаемого несколькими разработчиками.

PSR-2: Coding Style Guide
Этот стандарт расширяет вешеописанный.
Он призван уменьшить сложность восприятия кода, написанного разными авторами.
Это достигается путём внедрения правил относительно форматирования PHP-кода.

PSR-4: Autoloader
Данный стандарт описывает то,
как должна быть реализована автозагрузка классов
– что неймспейсы должны соответствовать директориям и ещё некоторые детали.
